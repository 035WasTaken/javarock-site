About JavaRock

JavaRock was started because I believe that Java Edition is superior, yet many of my friends only can play on Bedrock. I found a plugin called GeyserMC, and it allows for crossplay between the two editions (with the server still running on Java).

Season 4 is starting soon on Java 1.18, and we will be using the following plugins (as/if/when they are released):

  • GeyserMC - crossplay
  • Floodgate - crossplay
  • Cave Biomes - terrain
  • Crowmap - utility
  • Simpler Harvest - utility
  • Keep Head Names - utility
  • Mini MOTD - utility
  • SAX (1.17 branch) - utility
  • Styled Player List - utility
  • Lithium - performance
  • Starlight - performance
  • Fat Experience Orbs - performance
  • Fast Furnace - performance
  • World Specific View Distance - performance

as well as the following gamerules:

  • PlayersSleepingPercentage: 25%
  • SpawnRadius: 3
  • (All other gamerules are at their default values)

and some datapacks from Vanilla Tweaks:

  • AFK Display
  • Durability Ping
  • More Mob Heads
  • Player Head Drops
  • Silence Mobs
  • Track Raw Statistics
  • Track Statistics
  • Wandering Trades

In addition to this, we also have a custom datapack that selects a random event every week. This can be something like tag (you should know what this is) or wither hunt (kill as many winners as possible). In games with winners, they will be awarded, but in a game with losers, they will get punished.


Sadly, crossplay between 2 fundamentally different versions of the game is not (yet) perfected (and thus playing on the Java Edition is a better experience). It does still work well, but these are the most prevalent issues:

  • On mobile, buckets only place in the middle of the screen as the Java Edition packet doesn't specify where the bucket is placed
  • Bedrock players may occasionally experience rubberbanding when going over slabs, stairs, etc. as Bedrock treats their hitboxes differently
  • Bedrock players can't easily put things other than shields, totems, and arrows into their offhand as this is a client-side limitation. You can emote to swap your main hand and offhand, but you can't use your offhand items unless you can on Bedrock.
  • The glowing effect doesn't work and the spectral arrow texture is just a normal arrow as they don't exist in Bedrock
